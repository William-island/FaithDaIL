# Code For FaithDaIL
Welcome to the official code repository for our **NeurIPS 2025** accepted paper:  
**"Faithful Dynamic Imitation Learning from Human Intervention with Dynamic Regret Minimization."**

We will release detailed instructions for code deployment and usage soon. Stay tuned!


## Repository Structure

- `train_online.py` — Entry point for training and evaluation
- `algos/` — Model architectures
- `envs/` — Environment files
- `configs/` — Experiment configuration files for each simulator
- `logs/` — Output and logs
